FigmaToGFX

Convert Figma objects into code for TFT_eSPI bb_spi_lcd or AdafruitGFX with touch support

About FigmaToGFX

FigmaToGFX is a tool created by zynth for converting Figma objects into code compatible with TFT_eSPI bb_spi_lcd or AdafruitGFX with touch support.

For more information and to access the code, visit the GitHub repository.
